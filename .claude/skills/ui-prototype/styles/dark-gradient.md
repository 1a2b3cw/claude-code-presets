# 深色渐变

## 设计语言
深色背景、紫蓝渐变强调色、玻璃拟态卡片、霓虹感高亮。科技感、沉浸式、适合娱乐/工具类应用。

## 色彩系统
- 背景：主 `#0a0a0f` / 次 `#12121a` / 卡片 `rgba(255,255,255,0.05)`
- 文字：主 `#ffffff` / 副 `rgba(255,255,255,0.6)` / 辅助 `rgba(255,255,255,0.3)`
- 主色渐变：`linear-gradient(135deg, #7C3AED, #3B82F6)`（紫→蓝）
- 辅色：`#22d3ee`（青色高亮）
- 边框：`rgba(255,255,255,0.1)`
- 状态：成功 `#22c55e` / 警告 `#eab308` / 错误 `#ef4444`
- 图表：`#7C3AED` `#3B82F6` `#22d3ee` `#a78bfa` `#60a5fa`

## 字体
- 字体栈：`-apple-system, 'PingFang SC', 'Hiragino Sans GB', sans-serif`
- 大标题：24px / 700
- 标题：18px / 600
- 正文：15px / 400
- 辅助：13px / 400
- 标签：11px / 500

## 间距
- 页面边距：16px（左右）/ 20px（上下）
- 内容区间距：12-16px
- 卡片内边距：16-20px

## 圆角
- 卡片：16-20px
- 按钮：12px
- 输入框：12px
- 胶囊标签：20px
- 图标背景：12-14px

## 阴影
- 卡片：`0 4px 24px rgba(0,0,0,0.3)`
- 悬浮：`0 8px 32px rgba(124,58,237,0.2)`
- 发光效果：`0 0 20px rgba(124,58,237,0.3)`

## 组件规范

### 按钮
- 主按钮：紫蓝渐变底 + 白字，圆角 12px，高度 44-48px
- 次按钮：透明底 + 白字 + 1px `rgba(255,255,255,0.2)` 边框
- 文字按钮：无边框无背景，白色文字
- 禁用：`rgba(255,255,255,0.05)` 底 + `rgba(255,255,255,0.3)` 字

### 卡片
- 玻璃拟态：`background: rgba(255,255,255,0.05)` + `backdrop-filter: blur(12px)`
- 圆角 16-20px，边框 `1px solid rgba(255,255,255,0.1)`
- 内边距 16-20px
- 标题 16-18px / 600 白色，正文 14-15px / 400 白色60%透明

### 输入框
- `rgba(255,255,255,0.05)` 底，圆角 12px，高度 44px
- 边框 `1px solid rgba(255,255,255,0.1)`
- 焦点态：边框变为渐变色或 `#7C3AED`
- placeholder 颜色 `rgba(255,255,255,0.3)`

### 导航
- 顶部导航：`rgba(10,10,15,0.8)` + `backdrop-filter: blur(12px)`
- 底部 Tab：同上，激活态渐变色图标，未激活 `rgba(255,255,255,0.4)`
- 侧边栏：深色背景 + 玻璃拟态

### 列表项
- 高度 56-64px，左右 padding 16px
- 分隔线：`rgba(255,255,255,0.06)`
- 点击态：背景变为 `rgba(255,255,255,0.08)`

## 装饰元素
- 渐变光晕：大面积紫蓝渐变模糊圆，用于背景装饰
- 网格线：`rgba(255,255,255,0.03)` 细线网格
- 渐变文字：标题可用渐变 `background-clip: text`

## 禁止项
- 纯白背景
- 浅色卡片在深色背景上（用玻璃拟态代替）
- 高饱和度暖色大面积使用
- 过于复杂的渐变（最多 2 色渐变）
- 重阴影（用发光效果代替）
