# Apple 简约

## 设计语言
白底、黑白灰为主、克制用色、大留白、通过字号和字重区分层级。精致、克制、高可读性。

## 色彩系统
- 背景：主 `#ffffff` / 次 `#f5f5f7`
- 文字：主 `#1d1d1f` / 副 `#86868b` / 辅助 `#aeaeb2`
- 主色：`#1d1d1f`（深黑，用于按钮和强调）
- 辅色：无（不使用彩色作主按钮）
- 边框：`#e5e5ea`
- 状态：成功 `#34c759` / 警告 `#ff9f0a` / 错误 `#ff3b30`
- 图表：`#5e5ce6` `#64d2ff` `#30d158` `#ff9f0a` `#ff375f`（柔和色系）

## 字体
- 字体栈：`-apple-system, 'PingFang SC', 'Hiragino Sans GB', sans-serif`
- 大标题：24px / 700
- 标题：18px / 600
- 正文：15px / 400
- 辅助：13px / 400
- 标签：11px / 500

## 间距
- 页面边距：16px（左右）/ 20px（上下）
- 内容区间距：12-16px
- 卡片内边距：14-20px

## 圆角
- 卡片：16px
- 按钮：12px
- 输入框：12px
- 胶囊标签：20px
- 图标背景：10-14px

## 阴影
- 卡片：`0 1px 3px rgba(0,0,0,0.04)`
- 悬浮：`0 4px 12px rgba(0,0,0,0.08)`
- 不使用边框+阴影同时出现

## 组件规范

### 按钮
- 主按钮：`#1d1d1f` 深黑底 + 白字，圆角 12px，高度 44-48px
- 次按钮：白底 + `#1d1d1f` 字 + 1.5px 边框
- 文字按钮：无边框无背景，`#1d1d1f` 文字
- 禁用：`#f5f5f7` 底 + `#aeaeb2` 字

### 卡片
- 白底 `#ffffff`，圆角 16px，阴影 `0 1px 3px rgba(0,0,0,0.04)`
- 内边距 14-20px
- 标题 16-18px / 600，正文 14-15px / 400

### 输入框
- 白底或 `#f5f5f7` 浅灰底，圆角 12px，高度 44px
- 焦点态：边框色变为 `#1d1d1f`
- placeholder 颜色 `#aeaeb2`

### 导航
- 顶部导航：白底，底部分隔线 `#e5e5ea`
- 底部 Tab：白底，激活态 `#1d1d1f`，未激活 `#aeaeb2`
- 侧边栏：白底或 `#f5f5f7`

### 列表项
- 高度 52-60px，左右 padding 16px
- 分隔线：`#e5e5ea`，1px，左侧对齐文字
- 点击态：背景变为 `#f5f5f7`

## 禁止项
- 紫色渐变白底背景
- 彩虹色齐上阵
- 高饱和度大面积使用
- 蓝色作为主按钮色
- 纯黑 `#000000` 做背景
- 使用 emoji 代替图标
- 弹跳、闪烁等花哨动效
- 毛玻璃 / 玻璃拟态效果
