# 新拟物

## 设计语言
纯白背景、粗黑边框、厚重偏移阴影、大胆色块、几何感强。复古游戏风、个性鲜明、辨识度高。

## 色彩系统
- 背景：`#ffffff` / `#fffef2`（微暖白）
- 文字：主 `#000000` / 副 `#555555`
- 主色：`#ff6b35`（橙红）
- 辅色：`#004e89`（深蓝）/ `#ffd166`（暖黄）
- 边框：`#000000`，2-3px 实线
- 状态：成功 `#06d6a0` / 警告 `#ffd166` / 错误 `#ef476f`
- 图表：`#ff6b35` `#004e89` `#ffd166` `#06d6a0` `#ef476f`

## 字体
- 字体栈：`'Space Mono', 'Courier New', monospace`（标题）/ `-apple-system, 'PingFang SC', sans-serif`（正文）
- 大标题：28px / 700
- 标题：20px / 700
- 正文：15px / 400
- 辅助：13px / 400
- 标签：12px / 700 / 大写

## 间距
- 页面边距：20px（左右）/ 24px（上下）
- 内容区间距：16px
- 卡片内边距：20px

## 圆角
- 卡片：0-4px（几乎直角）
- 按钮：0-4px
- 输入框：0-4px
- 标签：0px

## 阴影
- 卡片：`4px 4px 0px #000000`
- 悬浮：`6px 6px 0px #000000`
- 按钮点击：阴影消失，元素偏移到阴影位置

## 组件规范

### 按钮
- 主按钮：`#ff6b35` 底 + 白字 + 2px 黑边框 + `4px 4px 0px #000` 阴影
- 次按钮：白底 + 黑字 + 2px 黑边框 + 同上阴影
- 点击态：`transform: translate(2px, 2px)` + 阴影消失
- 高度 44-48px，padding 0 24px

### 卡片
- 白底，2-3px 黑边框，`4px 4px 0px #000` 阴影
- 圆角 0-4px，内边距 20px
- 可用色块做顶部装饰条（4-6px 高）

### 输入框
- 白底，2px 黑边框，圆角 0-4px，高度 44px
- 焦点态：边框色变为主色 `#ff6b35`
- placeholder 颜色 `#999`

### 导航
- 白底 + 底部 2px 黑边框
- 激活态：主色背景 + 白字
- 未激活：黑字

### 列表项
- 高度 56px，左右 padding 16px
- 分隔线：1px 黑色实线
- 悬浮态：主色背景 + 白字

## 禁止项
- 圆角大于 8px
- 渐变背景
- 毛玻璃效果
- 极淡阴影（阴影必须明显）
- 柔和配色（颜色要大胆）
