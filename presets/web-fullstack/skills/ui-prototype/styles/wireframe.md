# 极简线框

## 设计语言
灰白底、细线框、无阴影无渐变、纯排版驱动、内容为王。原型感强、适合快速验证、不抢内容风头。

## 色彩系统
- 背景：主 `#fafafa` / 次 `#ffffff`
- 文字：主 `#111111` / 副 `#666666` / 辅助 `#999999`
- 主色：`#111111`（纯黑）
- 辅色：`#0066ff`（链接蓝）
- 边框：`#e0e0e0`
- 状态：成功 `#22c55e` / 警告 `#f59e0b` / 错误 `#ef4444`
- 图表：`#111` `#666` `#999` `#ccc`（灰度为主）

## 字体
- 字体栈：`-apple-system, 'PingFang SC', 'Hiragino Sans GB', sans-serif`
- 大标题：22px / 600
- 标题：17px / 600
- 正文：15px / 400
- 辅助：13px / 400
- 标签：11px / 500

## 间距
- 页面边距：16px（左右）/ 20px（上下）
- 内容区间距：12px
- 卡片内边距：16px

## 圆角
- 卡片：8px
- 按钮：6px
- 输入框：6px
- 标签：4px

## 阴影
- 无阴影
- 层级靠边框和背景色区分

## 组件规范

### 按钮
- 主按钮：`#111111` 底 + 白字，圆角 6px，高度 40px
- 次按钮：白底 + `#111111` 字 + 1px `#e0e0e0` 边框
- 文字按钮：无边框无背景，`#0066ff` 文字
- 禁用：`#f5f5f5` 底 + `#ccc` 字

### 卡片
- 白底 `#ffffff`，1px `#e0e0e0` 边框，圆角 8px
- 无阴影，内边距 16px
- 标题 16px / 600，正文 14px / 400

### 输入框
- 白底，1px `#e0e0e0` 边框，圆角 6px，高度 40px
- 焦点态：边框色变为 `#111111`
- placeholder 颜色 `#999`

### 导航
- 白底 + 底部 1px `#e0e0e0` 分隔
- 激活态：`#111111` 文字 + 底部 2px 黑线
- 未激活：`#999` 文字

### 列表项
- 高度 48-52px，左右 padding 16px
- 分隔线：1px `#e0e0e0`
- 无悬浮特效

## 禁止项
- 阴影（任何 box-shadow）
- 渐变（任何 gradient）
- 圆角大于 12px
- 装饰性元素
- 动效（除必要的 transition）
- 高饱和度颜色
